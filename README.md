# Quote Bot for Nerimity

A bot that generates quote images from messages.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and add your token:
```bash
cp .env.example .env
```

3. Run the bot:
```bash
npm start
```

## Usage

1. Mention the bot (e.g., `@YourBot`) in a channel
2. The bot will quote the message you're replying to 

## Features

- Generates quote images with user avatar on the right
- Dark theme with gradient background
- Shows quoted message, username, and bot name
